CatBot is dedicated to helping hunters in Monster Hunter World

"CatBot is a bot dedicated to providing resources on monsters in Monster Hunter World. It helps you with different aspects of the hunt such as the monsters weaknesses in certain states, what ailments affect it and where it can be found. CatBot can also assist with endemic life information, with a small description, and where it can be found."

Main

+mhw - MHW - Monster Hunter World: Iceborne

+mhgu - MHGU - Monster Hunter Generations Ultimate

+lfg - LFG - Looking For Group

+calc - Math/Calculation (MHWI)

General

+about - Shows extra information about the bot

+catfact - Shows a random cat fact

+changelog - Shows latest update log

+help - List all commands and their information

+invite - Invite CatBot to your Server

+support - Support server for CatBot

MHW - Monster Hunter World: Iceborne

+mhw armor [armor name] - Get info for a specific armor set

+mhw deco [deco name] - Get info for a specific decoration

+mhw events - Get all current events info

+mhw item [item name] - Get info for a specific item

+mhw list - List all monsters in MHW & Iceborne

+mhw monster [monster name] - Get info for a specific monster

+mhw rollhunt - Get a random roll of what monster you should hunt with which gear

+mhw skill [skill name] - Get info for a specific skill

+mhw weapon [weapon name] - Get info for a specific weapon

Monster Hunter Generations

+mhgu monster [monster name] - Get monster info

+mhgu weapon [weapon name] - Get weapon info

Monster Hunter Math

+calc dmgtaken [defense] - Percentage of damage taken based on defense

+calc elem [sharpness] [true elem] (monster hitzone) - Effective elemental attack (average damage over time)

+calc eraw [sharpness] [true raw] [critical boost level] (monster hitzone) - Effective raw attack (average damage over time)

+calc trueelem [attack] - True elemental value (removed bloat modifier)

+calc trueraw [weapon type] [attack] - True attack value (removed bloat modifier)

Looking For Group

+lfg cancel - Cancel your current active session

+lfg find - Show a menu listing all of the current active user sessions

+lfg post [platform] [session id] (description) - Posts an active session to CatBots LFG command

+lfg subscribe (channel name) - All user posted sessions will be sent to the subscribed Discord channel
